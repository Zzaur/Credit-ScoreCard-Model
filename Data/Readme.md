Overview
This directory contains the datasets used for the Credit Scoring project. The data originates from Lending Club and contains loan data for all loans issued from 2007 to 2018Q4.
File Descriptions
raw/accepted_2007_to_2018Q4.csv.gz
The primary dataset containing the complete loan data.

Source: Lending Club (Kaggle)
Size: ~1.6GB (uncompressed)
Rows: ~2.26 Million
Columns: 151 features
Target Variable: derived from loan_status
